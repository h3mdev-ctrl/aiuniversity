# Spaced Repetition

**One-liner:** Reviewing material at expanding intervals beats cramming, because
retrieving something just before you'd forget it is what strengthens the memory --
the core mechanism behind [[active-recall]] and tools like [[anki]].

## Why it works

- Memory strengthens most when retrieved at the edge of forgetting.
- Expanding intervals (1d -> 3d -> 1w -> 1m) track the forgetting curve, so you
  spend effort only when a review actually reinforces.

## How to apply

1. Turn what you learn into question -> answer pairs.
2. Review on a schedule that widens each time recall succeeds.
