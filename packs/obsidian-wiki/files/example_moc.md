# Learning Techniques -- Map of Content

**One-liner:** Single anchor for "how do I actually learn and retain things?" --
routes to the techniques that work, what they're for, and the order to apply them.
A MOC like this is what turns a folder of notes into something navigable.

## The techniques

| Technique | What it does | Note |
|---|---|---|
| Active recall | Test yourself instead of re-reading | [[active-recall]] |
| Spaced repetition | Review at expanding intervals | [[spaced-repetition]] |
| Interleaving | Mix topics instead of blocking one | [[interleaving]] |

## Where to start

New to this? Build [[active-recall]] first, then layer [[spaced-repetition]] on
top of it. [[interleaving]] comes once the first two are a habit.
