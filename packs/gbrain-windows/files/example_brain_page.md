# concepts/spaced-repetition   (EXAMPLE brain page -- learn the shape)

**One-liner:** Reviewing at expanding intervals beats cramming -- retrieval near the
point of forgetting is what strengthens memory.

## Key points
- Memory strengthens most when retrieved at the edge of forgetting.
- Expanding intervals (1d -> 3d -> 1w -> 1m) track the forgetting curve.
- Pairs with active recall: the recall attempt IS the strengthening event.
